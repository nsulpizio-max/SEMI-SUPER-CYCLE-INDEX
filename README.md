# Super Cycle Index Tracker

Mobile-friendly Next.js app for tracking the Semiconductor Super Cycle Pressure Index.

## Features
- Weekly index tracker
- Trend chart
- Definitions
- Key events column
- News Feed Aggregator
- Vercel serverless RSS news endpoint

## Deploy to Vercel
1. Go to https://vercel.com/new
2. Import this project folder from GitHub, or drag/drop after uploading to a repo.
3. Use default settings:
   - Framework: Next.js
   - Build command: npm run build
   - Output: .next
4. Deploy.
5. Open the Vercel URL on your iPhone.
6. Tap Share > Add to Home Screen.

## Local Run
npm install
npm run dev
